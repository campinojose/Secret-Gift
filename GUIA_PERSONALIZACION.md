# Guía de Personalización - Nuestra Historia

¡Felicidades por tu 4º aniversario! He creado esta página pensando en cada detalle para ustedes. Aquí tienes cómo personalizarla:

## 1. Fotos
Reemplaza las imágenes en la carpeta `public` o añade nuevas.
- Edita `components/sections/Gallery.tsx` para cambiar las rutas de las fotos y los textos.
- Puedes cambiar el fondo principal reemplazando `public/hero-bg.png`.

## 2. Música
Añade tu canción favorita:
- Pon un archivo MP3 en la carpeta `public` llamado `song.mp3`.
- O edita `components/ui/MusicPlayer.tsx` para cambiar el nombre del archivo.

## 3. Textos
- Historia principal: Edita `components/sections/Hero.tsx`.
- Hitos del Timeline: Edita `components/sections/Timeline.tsx`.
- Mensaje final: Edita `app/page.tsx` al final del archivo.

## Ejecutar
Para ver la página mientras editas:
```bash
npm run dev
```

¡Espero que le encante! ❤️
